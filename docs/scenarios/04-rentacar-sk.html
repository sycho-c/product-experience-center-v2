C Before · 개인 카톡 영업 limitations

200명 영업사원이 각자 개인 카톡으로 고객을 응대하면 회사 어디에도 메타가 남지 않는다. 이직하면 고객도 같이 사라진다.

Before

개인 카톡 영업의 한계

외부 채널 응대 100%, 메타 적재 0%, 이직 시 고객 보존 0%.

  • 박찬호와 정대리가 개인 카카오톡에서만 소통 — 회사 시스템 외부
  • 견적서·사업자등록증을 개인 카톡에서만 — DB Mart 적재 0건
  • 김주임 이직 시 개인 카톡 고객 12명 회수 불가 (거래처 경쟁사 유출)
  • 두 달 뒤 재문의 시 사업자번호·차량용도 등 동일 정보 다시 요청
