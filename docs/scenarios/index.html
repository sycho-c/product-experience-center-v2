Scenario #01 · 보험

하나손해보험 GA 협업 채널

손글씨 사진과 카톡 텍스트를 OCR/NER로 자동 추출해 한 채널 안에서 설계까지 완결.

🏥 보험사 💬 다중 1:1 방 ⏱ 7단계
  • OCR / NER 자동 추출 (사진 + 텍스트 동시)
  • 여러 메시지 → 한 할 일 묶음 등록
  • SFA 고객 등록 → 동의서 → PDF 자동 회신

Scenario #02 · 금융/캐피탈

우리금융캐피탈 강남BIZ AG 단체방

70명 모집인 단체방 안에서 비밀 메시지로 내부 검토를 보호하고, 비즈폼·승인까지 완결.

🏦 금융 💬 70명 단체방 ⏱ 6단계
  • 같은 AG 70명 단체방 · "AG명 / 이름" 식별
  • 비밀 메시지로 내부 검토 노출 차단
  • 비즈폼 · 신용평가 · 승인 워크플로우 자동 추적

Scenario #03 · 제조/전자부품

가온전선 SalesBridge

파일을 회사 자산으로 자동 적재하고, 자재번호 한 번 검색으로 두 달 전 견적을 그대로 재활용.

🏭 제조 💬 다중 1:1 방 ⏱ 6단계
  • 모든 첨부 파일 자동 자산화 (개인 단말 의존 제거)
  • 자재번호 통합 검색 — 대화·파일 한 번에 조회
  • 재발주 시 과거 견적 그대로 재첨부 → 응대 95% 단축
src/scenarios/gaon-cable-1to1/ 체험하기
Scenario #04 · 렌터카/서비스

SK렌터카 법인폰 SalesBridge

200명 영업사원이 공식 카카오 상담톡으로 응대 · DB Mart 자산화 · 이직 시 12개 방 일괄 이관.

🚗 렌터카 💬 다중 1:1 방 ⏱ 7단계
  • 비즈뿌리오 알림톡 → 카카오 상담톡 공식 채널
  • FCM 푸시로 외근 중 즉시 응대 + 비즈폼 자산화
  • 관리자가 영업사원 이직 시 12개 방 일괄 이관
