C Before · 카카오톡으로 일할 때 limitations

설계사가 종이 메모를 사진으로 보내고, 매니저는 매번 정보를 수동으로 옮긴다. 같은 고객의 요청이 여러 카톡방에 분산된다.

Before

카카오톡 일상 업무의 한계

개인 카톡에 흩어진 비정형 정보, 매니저의 손으로 다시 옮겨야 하는 반복 작업.

  • 종이에 적힌 고객정보를 사진으로 받음 — 시스템 자동 입력 불가
  • 설계매니저가 외부 메모장에 직접 다시 적어 외부 시스템으로 옮김
  • 같은 고객의 요청이 여러 카톡방에 분산 — 통합 이력 추적 불가
  • 알림톡·고객 등록·동의서·설계서가 각각 다른 시스템·다른 채널
